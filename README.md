### 2. 麦肯锡七步问题解决法（7-Step Problem Solving）

麦肯锡经典问题解决框架：

1. **定义问题**（Define the Problem）
2. **分解问题**（Break the Problem into Components）
3. **优先级排序**（Prioritize the Components）
4. **制定工作计划**（Develop a Workplan）
5. **分析问题**（Analyze the Components）
6. **综合结论**（Synthesize the Findings）
7. **呈现建议**（Present the Recommendations）

**GitHub 开源实现：**
- https://github.com/topics/mckinsey （7-step framework as Claude Code skill）

### 3. 麦肯锡方法论（McKinsey Way）
